# Max Bot API Schemas

Архив OpenAPI-схем для [Max Bot API](https://dev.max.ru/docs-api).

Все схемы взяты из открытого доступа с официального сайта документации: **https://dev.max.ru/docs-api**

## Назначение

Репозиторий позволяет отслеживать изменения в API Max между версиями — делать `diff` схем, автоматически генерировать
клиентские библиотеки или документацию на основе исторических снимков.

## Конвертор JSON → YAML

В репозитории есть конвертор `json2yaml.js`, который превращает JSON-схему в YAML
в формате Swagger/OpenAPI. Он построен на библиотеке [`js-yaml`](https://github.com/nodeca/js-yaml)
с настройками по умолчанию (отступ 2 пробела, перенос строк на 80 символов,
одинарные кавычки).

### Требования

- [Node.js](https://nodejs.org/) версии 14 или новее (включает `npm`).

### Установка

Из корня репозитория выполните один раз:

```bash
npm install
```

Команда скачает зависимость `js-yaml` в каталог `node_modules/` (он исключён из
гита через `.gitignore`).

### Конвертирование

Конвертировать один файл (выходной путь по умолчанию — то же имя с расширением `.yaml`):

```bash
node json2yaml.js schema_2026_06_02.json
# -> schema_2026_06_02.yaml
```

Указать выходной путь явно:

```bash
node json2yaml.js schema_2026_06_02.json schema_2026_06_02.yaml
```

Конвертировать сразу все файлы `schema_*.json` в текущем каталоге (запуск без аргументов):

```bash
node json2yaml.js
```

Эквивалентная команда через npm-скрипт:

```bash
npm run convert -- schema_2026_06_02.json
```
