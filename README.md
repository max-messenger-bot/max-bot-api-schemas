# Max Bot API Schemas

Архив OpenAPI-схем для [Max Bot API](https://dev.max.ru/docs-api).

Все схемы взяты из открытого доступа с официального сайта документации: **https://dev.max.ru/docs-api**

## Содержимое

| Файл                               |  Дата снимка   | Версия API |   Формат    |
|------------------------------------|:--------------:|:----------:|:-----------:|
| `schema_2026_04_15.yaml` / `.json` | 15 апреля 2026 |   0.0.1    | OpenAPI 3.0 |
| `schema_2026_04_30.yaml` / `.json` | 30 апреля 2026 |   0.0.1    | OpenAPI 3.0 |
| `schema_2026_05_07.yaml` / `.json` |   7 мая 2026   |   0.0.30   | OpenAPI 3.0 |
| `schema_2026_05_20.yaml` / `.json` |  20 мая 2026   |   0.0.30   | OpenAPI 3.0 |

Каждая схема представлена в двух форматах: YAML и JSON.

## Назначение

Репозиторий позволяет отслеживать изменения в API Max между версиями — делать `diff` схем, автоматически генерировать
клиентские библиотеки или документацию на основе исторических снимков.
